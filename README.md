🎮 Tic Tac Toe – Java GUI Game
📌 Project Description

This project is a Tic-Tac-Toe desktop application developed using Java with a graphical interface built using Java Swing.

The game allows two players to play the classic Tic-Tac-Toe game on a 3×3 grid. Players take turns placing X and O until one player wins or the game ends in a draw.

This project demonstrates fundamental concepts of GUI development, event handling, and game logic implementation in Java.

✨ Features
Interactive 3×3 game board
Two-player gameplay
Automatic turn switching (X and O)
Winner detection
Draw detection
Simple and user-friendly interface
🛠 Technologies Used
Java
Java Swing
Object-Oriented Programming (OOP)
▶️ How to Run the Project
Clone this repository or download the project files.
Open the project in a Java IDE such as IntelliJ IDEA, Eclipse (IDE), or NetBeans.
Compile and run the main Java file.
The Tic-Tac-Toe game window will open.
